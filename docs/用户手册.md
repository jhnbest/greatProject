# NVR设备监控系统 - 用户操作手册

## 1. 系统概述

NVR设备监控系统用于实时监控集团公司的视频监控NVR存储设备，支持海康威视和大华两大品牌的设备集成，提供设备状态监控、硬盘使用率统计、异常告警等功能。

## 2. 登录系统

### 2.1 访问地址

在浏览器中输入系统地址，打开登录页面。

### 2.2 输入账号密码

- **用户名**: 您的登录账号
- **密码**: 您的登录密码

### 2.3 登录成功

登录成功后，系统将自动跳转到仪表盘页面。

## 3. 仪表盘

仪表盘是系统的首页，展示系统的整体运行状况。

### 3.1 统计卡片

页面顶部展示四个核心统计卡片：

| 卡片 | 说明 |
|------|------|
| 设备总数 | 当前系统中的设备总数 |
| 在线设备 | 当前在线的设备数量 |
| 离线设备 | 当前离线的设备数量 |
| 待处理告警 | 当前待处理的告警数量 |

### 3.2 数据图表

仪表盘包含以下数据可视化图表：

- **硬盘使用率分布**: 展示各设备硬盘使用率的分布情况
- **设备在线状态**: 展示在线设备与离线设备的比例
- **近7天告警趋势**: 展示近7天的告警数量变化趋势
- **品牌分布**: 展示各品牌设备的数量占比
- **最新告警**: 展示最近发生的告警信息

## 4. 实时监控

### 4.1 监控列表

在实时监控页面，可以查看所有设备的实时状态。

### 4.1.1 筛选设备

使用筛选栏可以按以下条件筛选设备：

| 筛选条件 | 说明 |
|----------|------|
| 分公司 | 选择特定分公司查看 |
| 设备品牌 | 按海康/大华筛选 |
| 在线状态 | 按在线/离线筛选 |

### 4.1.2 监控指标

监控列表展示以下指标：

| 指标 | 说明 |
|------|------|
| 设备名称 | 设备的自定义名称 |
| IP地址 | 设备的网络地址 |
| 品牌 | 设备品牌标识 |
| 所属公司 | 设备所属分公司 |
| 在线状态 | 在线/离线状态 |
| 硬盘使用率 | 硬盘空间使用百分比 |
| 通道状态 | 在线通道数/总通道数 |
| 录像状态 | 录制中/停止 |

### 4.1.3 操作按钮

| 操作 | 说明 |
|------|------|
| 采集 | 手动触发设备数据采集 |
| 详情 | 查看设备详细信息 |

### 4.2 全量采集

点击"全量采集"按钮，系统将立即采集所有设备的状态数据。

## 5. 设备管理

### 5.1 查看设备列表

在设备管理页面，可以查看和管理所有NVR设备。

### 5.2 添加设备

1. 点击"添加设备"按钮
2. 填写设备信息：
   - **设备名称**: 自定义名称
   - **设备编码**: 唯一标识（不可重复）
   - **设备品牌**: 海康威视/大华
   - **设备型号**: 可选
   - **IP地址**: 设备的网络地址
   - **端口号**: API访问端口（默认8000）
   - **用户名**: 设备管理用户名
   - **密码**: 设备管理密码
   - **所属公司**: 选择所属分公司
   - **通道数量**: 设备的通道数量
   - **备注**: 可选备注信息
3. 点击"确定"保存

### 5.3 编辑设备

1. 在设备列表中找到要编辑的设备
2. 点击"编辑"按钮
3. 修改设备信息
4. 点击"确定"保存

### 5.4 测试连接

在编辑页面或设备列表中，点击"测试"按钮可以测试与设备的连接是否正常。

### 5.5 删除设备

1. 在设备列表中找到要删除的设备
2. 点击"删除"按钮
3. 在确认弹窗中点击"确定"

**注意**: 删除设备将同时删除该设备的历史数据和告警记录。

### 5.6 启用/禁用设备

通过设备状态开关可以启用或禁用设备。禁用的设备将不会参与数据采集和监控。

## 6. 设备详情

### 6.1 基本信息

展示设备的基本信息，包括：

- 设备编码、IP地址、品牌型号
- 所属公司、通道数量
- 最后在线/离线时间

### 6.2 硬盘信息

展示设备的硬盘使用情况：

- 硬盘总容量
- 已使用空间
- 剩余空间
- 使用率百分比

### 6.3 设备状态

展示设备的实时状态：

- 通道状态
- 录像状态
- 网络状态

### 6.4 趋势图表

展示设备硬盘使用率的24小时变化趋势。

## 7. 分公司管理

### 7.1 查看分公司列表

在分公司管理页面，可以查看所有分公司及其层级关系。

### 7.2 添加分公司

1. 点击"添加分公司"按钮
2. 填写分公司信息：
   - **公司名称**: 分公司名称
   - **公司编码**: 唯一标识
   - **上级公司**: 选择上级公司（留空为顶级）
   - **联系地址**: 分公司地址
   - **联系人**: 联系人姓名
   - **联系电话**: 联系电话
   - **备注**: 可选备注
3. 点击"确定"保存

### 7.3 添加下级公司

在分公司列表中，可以为没有下级的公司添加下级公司。

### 7.4 编辑/删除分公司

支持对分公司进行编辑和删除操作。

**注意**: 存在设备或子级公司的分公司无法删除。

## 8. 告警管理

### 8.1 告警列表

在告警管理页面，可以查看和管理所有告警信息。

### 8.1.1 筛选告警

使用筛选栏可以按以下条件筛选告警：

| 筛选条件 | 说明 |
|----------|------|
| 分公司 | 按分公司筛选 |
| 告警类型 | 按类型筛选（硬盘/设备/网络/通道） |
| 告警级别 | 按级别筛选（高危/中危/低危/信息） |
| 处理状态 | 按状态筛选（待处理/已处理） |
| 时间范围 | 按发生时间筛选 |

### 8.1.2 告警级别

| 级别 | 颜色 | 说明 |
|------|------|------|
| 高危 | 红色 | 需要立即处理的严重问题 |
| 中危 | 橙色 | 需要关注的问题 |
| 低危 | 灰色 | 一般性问题 |
| 信息 | 灰色 | 提示性信息 |

### 8.2 处理告警

1. 找到需要处理的告警
2. 点击"处理"按钮
3. 填写处理意见
4. 点击"确定"

### 8.3 批量处理

1. 在告警列表中勾选要处理的告警
2. 点击"批量处理"按钮
3. 填写处理意见
4. 点击"确定"

### 8.4 查看告警详情

点击告警标题或"详情"按钮，可以查看告警的详细信息。

### 8.5 待处理告警

点击顶部"待处理告警"按钮，可以快速筛选出所有待处理的告警。

## 9. 个人设置

### 9.1 基本信息

在"基本信息"标签页，可以查看和修改个人信息：

## 10. 常见问题

### Q1: 设备显示离线怎么办？

1. 检查设备网络是否正常
2. 检查设备IP地址是否正确
3. 检查设备是否开启API服务
4. 检查用户名和密码是否正确
5. 尝试在设备列表中点击"测试"按钮

### Q2: 硬盘使用率不更新怎么办？

1. 点击设备操作栏的"采集"按钮
2. 等待数据采集完成
3. 刷新页面查看最新数据

### Q3: 收不到告警通知怎么办？

1. 检查浏览器是否允许通知
2. 检查WebSocket连接是否正常
3. 查看告警列表是否有新告警

### Q4: 登录提示密码错误怎么办？

1. 确认用户名和密码是否正确
2. 联系管理员重置密码
3. 检查键盘大小写是否正确
